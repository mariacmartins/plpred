Exploratory Data Analysis¶

O plpred é uma aplicação de predição de localização subcelular de proteínas a partir da sua composição de aminoácidos.

Esse notebook refere-se à análise exploratória dos dados da aplicação plpred. De modo geral, a análise exploratória de dados é uma abordagem à análise de conjuntos de dados de modo a resumir suas características principais, frequentemente com métodos visuais.

from Bio import SeqIO
from Bio.SeqUtils import ProtParam
import pandas as pd
In [3]:
def compute_aa_composition(protein_sequence):
    analyzer = ProtParam.ProteinAnalysis(str(protein_sequence))
    return analyzer.get_amino_acids_percent()
In [4]:
#criando um campo “membrane” no df: 1 para as proteínas de membrana e 0 para as proteínas citoplasmáticas. 
df = pd.DataFrame()

handle = open('../data/raw/membrane.fasta')
parser = SeqIO.parse(handle, 'fasta')

for protein in parser:
    protein_data = compute_aa_composition(protein.seq)
    protein_data['membrane'] = 1
    df = df.append([protein_data], ignore_index=True)
    
handle = open('../data/raw/cytoplasm.fasta')
parser = SeqIO.parse(handle, 'fasta')

for protein in parser:
    protein_data = compute_aa_composition(protein.seq)
    protein_data['membrane'] = 0
    df = df.append([protein_data], ignore_index=True)

Características físico-químicas: Proteínas de membrana possuem características mais hidrofóbicas para interagir com ácidos graxos em relação à proteínas citoplasmáticas.

In [8]:
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
pca.fit(df.drop(['membrane'], axis=1))

for group, df_group in df.groupby('membrane'):
    components = pca.transform(df_group.drop('membrane', axis=1))
    plt.scatter(
        components[:,0],
        components[:,1],
        s=0.5,
        label='Membrane' if group == 0 else 'Cytoplasm',
        color='red' if group == 0 else 'blue'
    )

plt.show()

Análise de Componente Principal (Principal Component Analysis, PCA): Análise para ver, em duas dimensões, a diferença entre as proteínas de membrana e as proteínas citoplasmáticas.

A partir das características de composição de aminoácidos foi possível observar que houve grupos de proteínas que por conta das suas características são quase que exclusivamente proteínas de membranas e outras proteínas que são quase que exclusivamente proteínas de citoplasma. Esse resultado propõe um bom indicativo que o modelo conseguirá aprender a discriminar proteinas de membrana e citoplasma.

Métricas:

  • Acurácia: indica uma performance geral do modelo. Dentre todas as classificações, quantas o modelo classificou corretamente.
  • Precisão: dentre todas as classificações de classe positivo que o modelo fez, quantas estão corretas.
  • Recall: dentre todas as situações de classe positivo como valor esperado, quantas estão corretas.
  • F1-Score: média harmônica entre precisão e recall.
